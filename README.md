# Mouse-Recorder-with-python

## Mouse Recorder

![Mouse Recorder Demo](demo.gif)

این پروژه یک برنامه ساده برای ضبط و بازپخش حرکت و کلیک‌های موس است. این برنامه با استفاده از Python و Tkinter پیاده‌سازی شده است.

### نصب

برای استفاده از این برنامه، ابتدا باید پیش‌نیازهای آن را نصب کنید:

```bash
pip install pyautogui keyboard pynput
```

### استفاده

1. برای شروع ضبط حرکت موس، دکمه "Start Recording" را کلیک کنید.
2. حرکت و کلیک‌های موس خود را ضبط کنید. برای پایان ضبط، کلید `q` را فشار دهید یا دکمه "Stop" را کلیک کنید.
3. برای بازپخش حرکت و کلیک‌های ضبط شده، دکمه "Start Replaying" را کلیک کنید.
4. برای متوقف کردن پخش، کلید `q` را فشار دهید یا دکمه "Stop" را کلیک کنید.
